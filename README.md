getting started with flask
=====

This demo project consists of two entities, author and message.
Authors write messages. 

I hope to update this project to meet best practice guidelines.
However, we are not there yet. 

Please take a look at the code, 
    open some issues, 
    and submit a pull request if you like. 
     
No license yet because this is not very useful yet but maybe AGPL? 


how to run
==========

I assume you are on Ubuntu 16.04 Xenial. 
Details might be different if you're on a different distribution or os.

1. Prepare by installing these packages (globally?)

```apt-get install python3 python3-venv python3-dev libpq-dev build-essential```

2. Create a virtual environment for this project 

```buildoutcfg
python3 -m venv venv
```

You likely won't see any output for this 
but you will get a new folder called `venv` 
in your current directory. 

You can choose a different name for your folder. 
It is all good. 

3. Next, you will ask your terminal to use the virtual env

```buildoutcfg
source venv/bin/activate
```

Here's what you can expect: 

```buildoutcfg
root@a:~# source venv/bin/activate
(venv) root@a:~#
```

You should see a subtle difference in your terminal. 

Compare 

```buildoutcfg
root@a:~# 
```

and 

```buildoutcfg
(venv) root@a:~#
```

You will notice that the second option has a (venv) there. 
This is the virtual environment you are using.
